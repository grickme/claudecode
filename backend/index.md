# Backend Guides

Server-side development with Next.js API Routes, Python, and FastAPI.

## Guides

| Guide | Description |
|-------|-------------|
| [Local Development](./local-dev.md) | Run and test your app locally |
| [Next.js API](./nextjs-api.md) | API routes, middleware, authentication |
| [PDF Processing](./pdf-processing.md) | Extract content from PDFs (Marker, MinerU, Gemini) |
| [PDF Creation](./pdf-creation.md) | Generate PDFs (Playwright, jsPDF, WeasyPrint) |
| [FastAPI](./fastapi.md) | Python API framework |

## When to Use What

| Use Case | Technology |
|----------|------------|
| Full-stack web app | Next.js (frontend + API routes) |
| Read/extract PDF content | Python (Marker, MinerU) or Gemini Vision |
| Generate PDF reports | Playwright (HTML to PDF) or jsPDF |
| ML/data pipelines | Python (FastAPI) |
| Microservices | FastAPI or Cloud Functions |
