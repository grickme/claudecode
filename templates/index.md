# CLAUDE.md Templates

Ready-to-use templates for your project's Claude Code configuration.

## Templates

| Template | Description |
|----------|-------------|
| [Minimal](./claude-md-minimal.md) | Essential settings only |
| [Full](./claude-md-full.md) | Comprehensive with all sections |

## How to Use

1. Choose a template
2. Copy the content inside the code block
3. Create `.claude/CLAUDE.md` in your project
4. Paste and customize

## What Goes in CLAUDE.md

- Project overview and context
- Technology stack
- Common commands
- Project structure
- API documentation
- Environment variables
- Deployment instructions
- Troubleshooting tips

Claude Code automatically reads this file to understand your project.
